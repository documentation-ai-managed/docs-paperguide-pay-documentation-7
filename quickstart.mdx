## Prerequisites

Before you begin, ensure you have the following:

<Callout kind="info">

- A modern web browser like Chrome, Firefox, or Safari
- An active email address for account verification
- Optional: A GitHub account for version control integration

</Callout>

Paperguide-pay streamlines project documentation, letting you create organized spaces for teams to collaborate on guides, APIs, and resources.

## Create Your Account

Sign up takes under 2 minutes. Follow these steps to get started.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Open your browser and navigate to `https://app.paperguide-pay.com`.

    Click **Sign Up** in the top right corner.
  </Step>
  <Step title="Register" icon="user-plus">
    Enter your email, create a strong password, and provide your full name.

    Agree to the terms and click **Create Account**.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification email from Paperguide-pay.

    Click the link to activate your account.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the login page, enter your credentials, and access the dashboard.
  </Step>
</Steps>

## Create Your First Documentation Space

Once logged in, you land on the dashboard. Creating a space organizes your project docs.

<Steps>
  <Step title="New Space" icon="plus">
    Click **New Space** or the `+` button in the sidebar.
  </Step>
  <Step title="Configure Space" icon="settings">
    Name it (e.g., "Project Alpha Docs"), add a description like "Core documentation for Project Alpha", and select visibility: Public, Private, or Team.
  </Step>
  <Step title="Save and Enter" icon="check-circle">
    Click **Create**. You now see an empty space ready for content.
  </Step>
</Steps>

## Upload Initial Project Documents

Add your first files via multiple methods.

<Tabs>
  <Tab title="Drag & Drop" icon="upload">
    In your space, drag files from your desktop into the upload zone.

    Supported formats: Markdown (`.md`), PDFs, images (`.png`, `.jpg`), and ZIP archives.
  </Tab>
  <Tab title="URL Import" icon="globe">
    Paste a public URL (e.g., GitHub README).

    Paperguide-pay fetches and converts it automatically.
  </Tab>
  <Tab title="API Upload" icon="code">
    Use the Spaces API for programmatic uploads.

    <CodeGroup tabs="JavaScript,cURL">
      ````javascript
      const response = await fetch('https://api.example.com/v1/spaces/{spaceId}/documents', {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${YOUR_API_KEY}`,
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          name: 'project-overview.md',
          content: '# Project Overview\nYour content here.'
        })
      });
      ````

      ````bash
      curl -X POST https://api.example.com/v1/spaces/{spaceId}/documents \
        -H "Authorization: Bearer YOUR_API_KEY" \
        -H "Content-Type: application/json" \
        -d '{
          "name": "project-overview.md",
          "content": "# Project Overview\nYour content here."
        }'
      ````
    </CodeGroup>
  </Tab>
</Tabs>

## Basic Editing and Organization Tips

Edit pages using the visual editor or Markdown mode. Organize with folders and tags.

<ExpandableGroup>
  <Expandable title="Editing Basics" default-open="true">
    - Use the toolbar for headings, lists, and embeds
    - Preview changes in real-time
    - Save drafts automatically every 30 seconds
  </Expandable>
  <Expandable title="Organization Best Practices">
    - Create folders: `/api/`, `/guides/`, `/changelog/`
    - Add tags: `api`, `internal`, `draft`
    - Link pages with `[Internal Link](/path/to/page)`
  </Expandable>
</ExpandableGroup>
